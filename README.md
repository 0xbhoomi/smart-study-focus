# Smart Study Focus
Smart Study Focus is a Chrome/Edge extension that helps you stay focused while studying by blocking YouTube and Instagram. It is simple, easy to use, and improves your productivity.
## Features
- Blocks YouTube and Instagram with one click
- Easy to install on Chrome or Edge
- Helps you stay focused and avoid distractions
## Installation
### Option 1: Download from this repo
1. Click the green Code → Download ZIP button.  
2. Extract the folder.  
3. Open your browser:
   - Chrome: chrome://extensions
   - Edge: edge://extensions
4. Turn ON Developer mode (toggle at top right).  
5. Click Load unpacked → select the extracted folder.  
6. Enjoy focused study time!  
### Option 2: If you have the ZIP file
1. Download smart-study-focus.zip from this repo.  
2. Extract the folder.  
3. Follow steps 3–6 from Option 1.  
## How to Use
1. Open YouTube or Instagram.  
2. Click the Smart Study Focus extension icon in the browser.  
3. Click the Block button.  
4. The site will redirect and a message will appear to remind you to study.
